    }

11、自动装配
使用 @Autowired 进行自动装配，按照对象的类型进行自动装配
@Component("car")
public class Car {
    @Value("小黄车")
    private String name;
    
    @Value("blue")
    private String color;
}
@Component("person")
public class Person {
    @Value(value="helen")
    private String name;
    
    @Value(value="18")
    private Integer age;
    
    @Autowired //自动装配
    private Car car;
}

自动装配存在的问题：如果一个类型有多个对象，那么可以采用以下的方式
方式1：
使用@Qualifier指定具体的对象
    @Autowired
    @Qualifier("car2")
    private Car car;
方式2：
使用@Resource指定具体的对象
    @Resource(name="car1")
    private Car car;
